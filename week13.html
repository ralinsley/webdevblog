Week 13

I spend some more time looking over the official documentation for redux. They provide a handy catalog of large-scale projects that use the library so I perused some of the more interesting ones to explore what redux could create. I enjoyed looking at boardgame.io and seeing all the different tabletop games that people have implemented with redux.

type Profile = { profile: { currentGameId?: number } }
                type GFB = () => Object
                export const sendToFirebase = ({ profile }: Profile, getFirebase: GFB, action: string, node: string, data?: Object) => {
                  const { currentGameId } = profile
                  if (currentGameId) {
                    const firebase = getFirebase()
                    firebase[action](`/games/${currentGameId}/${node}`, data)
                  }
                }

(This is an example of one of many different types of actions)
There is even an implementation of an old board game called Axis and Allies! It's exciting to see the great variation of projects using react-redux.
